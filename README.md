# nerd-font-fira-code

Single family of fonts for trial (WIP).

## Install

On ubuntu:

```bash
$ cd
$ curl -LO https://github.com/sthagen/nerd-font-fira-code/raw/default/dist/FiraCode.zip && \
  unzip FiraCode.zip -d "$HOME"/.fonts && \
  fc-cache -fv
```
